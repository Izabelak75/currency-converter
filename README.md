# Currency Converter

![Currency Converter](images/small_share.png)

## Basic Description

Currency Converter is a **basic currency calculator** created as part of learning **Front-End Fundamentals**. 
It allows users to convert **PLN amounts to EUR, USD, and CAD** using predefined exchange rates. 
The language of the page is **Polish**.

## Functionality

- **Main Goal:** Build a simple interactive JavaScript application with a working form. 
- **Key Features:** 
  - Converts amounts from **PLN to EUR, USD, or CAD** 
  - Uses **static, predefined exchange rates** 
  - Result is **formatted** to display full calculation 
  - Implements **form handling using event listeners**, no manual submit required
